Comandos Úteis utilizados no Aircnc Rocketseat

-> ctrl + shift + p, install code, install code command path
Caso comando code . não esteja disponível

-> ctrl + shift + p, toogle
Toggle integrated terminal
ctrl + `

-> node src/server.js
Para executar servidor, abrir em localhosta:3333

-> yarn add nodemon -D
Para atualizar automaticamente o servidor ao salvar.
-D indica que é uma dependencia apenas de desenvolvimento.

-> Insomnia ou postman
Para testar requests.

- Aba Query pode realizar a o seguinte filtro
http://localhost:3333/users?idade=20
- Route params
http://localhost:3333/users/1

-> yarn add mongoose
Ferramenta que facilita a interação com mongodb

-> Acessar http://portquiz.net:27017/
Para saber se minha conexão está por traz de um proxy.
Se aparecer uma mensagem de texto sem erros, está ok minha conexão.
